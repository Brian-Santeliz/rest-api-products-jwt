## REST API (ES6+)

Technology:
*  Nodejs,
*  Express, 
* Mongoose, 
* Jwt, 
* Babel, 
* Mocha,
* Supertest

Download the project and use the next command:

Install dependencies:

`npm install`

Start Script: 

`npm start`

Build Project with Babel: 

`npm run build`

This command transpile the code using babel in a folder Build.

Development Script: 

`npm run dev`

This command execute babel-node & nodemon.
